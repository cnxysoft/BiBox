# Bili_UserNameUpdate

## 自动批量更新B站昵称

### 使用方法：
1.打开“ID列表.txt”，将Excel表格中的UID选中后覆盖粘贴在记事本中并保存。
2.打开“B站昵称获取工具.exe”，将保存好的“ID列表.txt”拖入程序并回车，等待程序执行完成。
3.打开“查询结果.txt”，全选复制其中数据并粘贴回Excel中（包含2列，分别是：UID、昵称）。

### 注意：
    为了安全起见，使用前建议检查文件数字签名，右键“B站昵称获取工具.exe”菜单，点击属性，
    确认最上方存在“数字签名”标签页即可。
